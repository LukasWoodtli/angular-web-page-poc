# Modes

| Key   | Mode   |
|-------|--------|
| `i`   | Insert |
| `ESC` | Normal |


# Navigation

| Key | Movement  |
|-----|-----------|
| `h` | ← (left)  |
| `j` | ↓ (down)  |
| `k` | ↑ (up)    |
| `l` | → (right) |

> Hint: `j` looks like an arrow pointing down

# Editing

| Key  | Action                         |
|------|--------------------------------|
| `x`  | Delete (cut) char under cursor |
| `dd` | Delete (cut) current line      |
| `p`  | Paste                          |


# Help

- `:help [command]`: Get help (without command: general help)
